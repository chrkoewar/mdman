# keepass

## documentation
[docu](https://www.mankier.com/1/keepassxc-cli)

## keys
- `ctrl+enter`  save entry

## cmd
```
keepassxc-cli search path/to/database_file name
keepass-cli clip /path/to/db /path/to/key
keepassxc-cli generate --lower --upper --numeric --special --length 16
keepassxc-cli add --generate path/to/database_file entry_name
sync keepass https://keepass.info/help/v2/sync.html file > synchronize
```
