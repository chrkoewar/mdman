# mocp

music player

## keys

### general
- `q`    detach from server
- `Q`    quit
- `S`    shuffle toggle
- `R`    repeat toggle
- `^t`   show length
- `,.`   volume
- `^d`   delete file
- `c`    copy to ${MusicDir}

### design
- `tab`  toggle playlist
- `l`    window layout
- `T`    themes

### navigation
- `i`    goto dir
- `o`    open URL
- `/`    search
- `m`    music directory
- `~`    fast dir 1
- `,`    fast dir 2
- `G`    active dir

### playlist / queue
- `V`    save playlist
- `a`    add to playlist
- `A`    add recursively
- `d`    delete from playlist
- `C`    clear playlist
- `u`    move item up
- `j`    move item down
- `z`    un/queue

## commands
```
mocp --server
mocp --clear
mocp --append path/to/directory
mocp --exit
```
