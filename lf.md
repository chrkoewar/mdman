# lf - file manager

## keys

- `Y`                   copy filename
- `v`                   invert selection
- `u`                   unselect
- `:`                   read (default)  builtin/custom command
- `$`                   shell           shell command
- `%`                   shell-pipe      shell command running with the UI
- `!`                   shell-wait      shell command waiting for a key press
- `&`                   shell-async     shell command running asynchronously
- `sn`                  sort natural
- `ldcd`                in bash
- `e`                   edit
- `E`                   edit with fzf
- `i`                   preview
- `y/p`
- `q`

## commands

- glob-select *txt
- glob-unselect
- :filter
- $f filename
- $fs selected files
- $fx fs || f
- $pwd
