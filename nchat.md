# nchat - WhatsApp messenger

## keys

- `^o` open
- `~o` open link
- `^e` edit
- `^x` send/reply
- `^f` forward
- `^r` react
- `^d` download
- `~d` save
- `^q` close/quit
- `A-c` copy msg

- `Tab` next chat
- `PageUp / PageDown`

- `^l` line break
- `^t` send file
- `^x` send
- `^j` emoji

- `^n` chats 
- `~n` contacts
- `A-u` unread

- `A-/` find
- `A-|` next hit

- `A+,/.` Breite
