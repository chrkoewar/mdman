# castero

listen to podcasts

## keys
- `1-5`   feeds / queue / downloads
- `enter` start episode
- `space` queue episode
- `p`     play/pause
- `a`     add feed
- `d`     delete feed
- `s`     save episode
- `d`     delete episode
- `f/b`   seek forward/back
- `u`     show url
- `e`     execute command
