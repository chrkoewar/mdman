## keys
- `%B,`  %d.%m.%Y
- `%B,`  %a, %F
- `^A`   add appointment
- `^T`   add todo
- `w/W`  next/prev week
- `m/M`  next/prev month
- `^G`   goto date
- `+/-`  priority
- `!`    flag
- `|`    pipe
- `a`    add
- `d`    delete
- `v`    view
- `n`    note    
- `TAB`  wechseln
- `^N`   Woche / Monat
- `^G`   Heute
